PlayerPositionPRO-

this plugin will make you able to save every player last position if they disconnect or they crash, completely customizable, and options to create automatic save with custom time.

Features:

  • Automatically save player position when he crashes or disconnect
  • Customizable: possibility to save last player position only in certain worlds
  • Create an automatic position saver (example: save player position every 'x' hours, or minutes)
  • It has an easy configuration system
  • (new features coming soon!)

Commands:

  • /pp reload , reloads the plugin.
  • /pp version , get info on plugin version and updates 
  • /pp help , a simple text to help who's using the plugin
  • /pp permissions , get the permissions list IN-GAME!

Permissions:

  • player.position.admin - makes you able to use all the PlayerPositionPro commands ( permissione reccomended only for admins & founders )

Tips:

You won't need to enable player.position.admin if you are already OP.

WARNING:[I Am not responsable if your server or this plugin crashes making your/your players lose all their items and their last position, this plugin won't save player position if server Crashes!]

Compatibility:

[This plugin will be tested in 1.9.4] This plugin may crash if you use not-tested versions!.
